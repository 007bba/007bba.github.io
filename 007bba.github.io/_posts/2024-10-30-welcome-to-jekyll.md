## 什么是 Jekyll？

Jekyll 是一个简单的静态网站生成器，它可以将 Markdown 文件转换为静态 HTML 网站。它的主要特点包括：

- **简单易用**：使用 Markdown 语法编写内容
- **版本控制友好**：所有文件都是纯文本
- **免费托管**：可以部署在 GitHub Pages 上
- **高度可定制**：支持主题和插件

## 为什么选择 Jekyll？

作为开发者，Jekyll 提供了很多优势：

1. **技术栈熟悉**：使用 Git 和 Markdown，这些都是开发者熟悉的工具
2. **性能优秀**：静态网站加载速度快
3. **安全性高**：没有数据库和动态脚本，减少了安全风险
4. **SEO 友好**：静态内容容易被搜索引擎收录

## 开始写作

要创建新的博客文章，只需要在 `_posts` 目录下创建一个新的 Markdown 文件，文件名格式为：

```
YYYY-MM-DD-标题.md
```

然后在文件开头添加 YAML 前置信息：

```yaml
---
layout: post
title: "文章标题"
date: 2024-10-30 10:00:00 +0800
categories: [分类]
---
```
